Liaison unidirectionnelle Interpolation

Affiche sur le template la valeur d'une variable du composant

Syntaxe : {{'{{'}}variable}}

Exemple :

Fichier TypeScript app.component.ts
export class monComposant {{'{'}}
      title : string = 'Angular Beginner Project';
{{'}'}}

Fichier app.component.html

<p>Le titre de mon application est : {{'{{'}} title }} </p>

Rendu :

Le titre de mon application est : {{title}}

 Property Binding

Lit la valeur d'une variable du composant et l'ajoute en tant que valeur de propriété sur un élément HTML

Syntaxe : [ propriete ] = "variable"

Exemple :

Fichier TypeScript app.component.ts
export class monComposant {{'{'}}
    title : string = 'Angular Beginner Project';
    valeurInput : string = 'une valeur venant du composant';
{{'}'}}

Fichier app.component.html

<input type="text" [value]="valeurInput" disabled/>

La valeur de input est automatiquement remplie avec la valeur de l'attribut "valeurInput" du composant.

Event Binding

Envoie des données au composant depuis un évènement sur le template

Syntaxe : {{'('}}evenement) = "methode()"

Exemple :

Fichier TypeScript app.component.ts
export class monComposant {{'{'}}
    title : string = 'Angular Beginner Project';
    valeurInput : string = 'une valeur venant du composant';
    info :string = '';
    onclickButton() : void {{'{'}}
      if(this.info == '') {{'{'}}
        this.info = 'Une information très intéressante';
      }else {{'{'}}
        this.info = '';
    }
}
          
Fichier app.component.html
 <button (click)="onclickButton()"> Afficher infos </button>
 <p>{{'{{'}}info}}</p>

Rendu :

{{info}}

 Liaison bidirectionnelle Two-ways-binding

Affiche et modifie dynamiquement la valeur de la variable du composant

Syntaxe : {{'[('}}ngModel)] = "variable" pour l'input, et {{'{{'}}variable}} pour l'affichage

Exemple :

Fichier TypeScript app.component.ts
export class monComposant {{'{'}}
    title : string = 'Angular Beginner Project';
    valeurInput : string = 'une valeur venant du composant';
    info :string = '';
    zoneSaisie : string ='Saisissez votre nom';
    onclickButton() : void {{'{'}}
        if(this.info == '') {{'{'}}
          this.info = 'Une information très intéressante';
        }else {{'{'}}
          this.info = '';
    }
}
            
Fichier  app.component.html

<input type="text" {{'[('}}ngModel)]="zoneSaisie"/>
<p> {{'{{'}}zoneSaisie}} </p>

Rendu :

Valeur de zoneSaisie : {{zoneSaisie}}
