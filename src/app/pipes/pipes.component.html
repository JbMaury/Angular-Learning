Les pipes Date

Formate une date suivant un schéma donné

Syntaxe : {{'{{'}} variableDate | date }}

La variable à formater peut être de type string / number / date
Par défaut, sans indication supplémentaire, le formatage se fait suivant la localisation de l'application.
Lors de sa construction, un projet Angular est localisé en en-US (English in the USA). Le formatage date donnera comme résultat pour la date du jour : {{dateJour | date }}

{{'{{'}} variableDate | date : 'dd/MM/yyyy' }} {{'{{'}} variableDate | date : 'HH:mm' }} {{'{{'}} variableDate | date : 'full' }} {{'{{'}} variableDate | date : 'short' }}

Des paramètres supplémentaires peuvent être ajoutées au pipe date pour préciser le format souhaité. Il est possible d'écrire ce format caractère par caractère, ou bien d'utiliser les raccourcis pré-conçus d'Angular :
short[Date/Time] / medium[Date/Time] / long[Date/Time] / full[Date/Time] .
La liste complète se trouve sur Angular.io.

Exemple :

Fichier TypeScript app.component.ts
  export class monComposant {{'{'}}
        title : string = 'Angular Beginner Project';
        dateJour : Date = new Date();
        monAnniversaire : Date = new Date('1984-03-01');
  {{'}'}}

Fichier app.component.html

(1)<p> {{'{{'}} monAnniversaire | date }} </p>

(2)<p> {{'{{'}} monAnniversaire | date : 'fullDate' }} </p>

(3)<p> {{'{{'}} monAnniversaire | date : 'dd/MM/yyyy HH:mm' }} </p>

Rendu (1) : {{monAnniversaire | date }}

Rendu (2) : {{monAnniversaire | date : 'fullDate' }}

Rendu (3) : {{monAnniversaire | date : 'dd/MM/yyyy HH:mm' }}

UPPERCASE/lowercase/TitleCase

Formate une chaîne de caractères en majuscule/minuscule

Syntaxe : {{'{{'}} variable | uppercase/lowercase/titlecase }}

La variable à formater doit être de type string

Exemple : 

Fichier TypeScript app.component.ts
  export class monComposant {{'{'}}
        title : string = 'Angular Beginner Project';
        dateJour : Date = new Date();
        monAnniversaire : Date = new Date('1984-03-01');
        identite : string = 'joachim maury';
        caseState : string = '{{caseState}}';


        toggleCase() : void {{'{'}}
          switch(this.caseState){{'{'}}
            case 'lower' : this.caseState = 'upper';
            break;
            case 'upper' : this.caseState = 'title';
            break;
            case 'title' : this.caseState = 'lower';
          }
        }
  }
          
Fichier app.component.html
<p [ngSwitch]="caseState" >
 <span> {{'{{'}} identite  | uppercase }} </span>
 <span> {{'{{'}} identite  | lowercase }} </span>
 <span> {{'{{'}} identite  | titlecase }} </span>
  <button (click)="toggleCase()"> Switch </button
</p>
Rendu :

{{identite | uppercase}} {{identite | lowercase}} {{identite | titlecase}}
